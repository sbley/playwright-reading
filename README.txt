# Read Assignments Confirmation

## Prerequisites

1. Check out this repository to a workspace on your computer.
2. Install `Node.js` on your computer, if you have not already.
3. Run `npm install` to install the required libraries.
4. Open `tests/read-assignments.spec.ts` and set your username.

## Run

1. Make sure you have read the updated content in the wiki :)
2. Run `npm test` to start the process. A browser will come up, and you can watch the run.
